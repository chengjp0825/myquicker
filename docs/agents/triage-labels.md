# Triage labels

This repo uses the default canonical triage label vocabulary.

| Role | Label string |
|------|--------------|
| Needs evaluation by a maintainer | `needs-triage` |
| Waiting on the reporter for more information | `needs-info` |
| Fully specified; ready for an AFK agent to pick up | `ready-for-agent` |
| Needs human implementation or decision | `ready-for-human` |
| Will not be actioned | `wontfix` |

Apply these labels as strings in the issue tracker. If this repo later moves to GitHub Issues, create labels with exactly these names.
